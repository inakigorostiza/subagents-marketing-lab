# Subagents Marketing Lab

A quick guide for marketing students on working with subagents in VS Code.

## About

This lab teaches you how to:
- Create reusable marketing specialists (copywriter, researcher, strategist)
- Use them directly in VS Code's integrated terminal (no tmux needed)
- Build a complete marketing workflow using subagents

## Quick Start

1. Visit: https://inakigorostiza.github.io/subagents-marketing-lab/
2. Follow the step-by-step guide
3. Set up your first subagent in VS Code

## Prerequisites

- Claude Code 2.1.32+
- VS Code

## Duration

- Setup: 10 minutes
- Creating subagents: 15-20 minutes
- Running a workflow: 30-45 minutes
- **Total: ~1 hour**

## Tech Stack

- HTML5, CSS3
- Claude Code (Subagents API)
- VS Code integrated terminal

## Key Topics

- What are subagents?
- Subagents vs Agent Teams
- Interactive subagent creation
- Manual markdown-based creation
- Workflow execution
- Best practices for marketing specialists
